# R Data Analysis Project

This repository contains mobile money analysis project developed using R and R Markdown.
The project focus on time-series analysis, anomaly detection, and data visualization
using publicly available Kenyan datasets.

## Project

###Mobile Money Ecosystem Analysis (2007–2023)
Analysis of Kenya’s mobile money agent activity, transaction volumes, and values,
highlighting long-term growth patterns and anomalous periods.

## Tools Used
- R
- R Markdown
- tidyverse
- ggplot2
- lubridate
